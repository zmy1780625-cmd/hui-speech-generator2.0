# 安装指南 - 晖总演讲稿生成器 V3.0

## 🎯 系统要求

### 必需环境
- **Node.js**: 14.0.0 或更高版本
- **npm**: Node.js自带的包管理器
- **现代浏览器**: Chrome、Firefox、Safari、Edge

## 📥 Node.js 安装

### Windows 系统
1. 访问 [Node.js官网](https://nodejs.org/)
2. 下载 LTS 版本（推荐）
3. 运行安装程序，按默认设置安装
4. 重启命令提示符或PowerShell

### 验证安装
打开命令提示符，执行以下命令：
```bash
node --version
npm --version
```

如果显示版本号，说明安装成功。

## 🚀 项目启动

### 方法一：一键启动（推荐）
1. 双击 `start.bat` 文件
2. 等待依赖安装完成
3. 服务器自动启动

### 方法二：手动启动
1. 打开命令提示符
2. 进入项目目录：
   ```bash
   cd "c:\Users\v_mykzhang\CodeBuddy\20260211155927\新的智能体\docs\speech-generaor"
   ```
3. 安装依赖：
   ```bash
   npm install
   ```
4. 启动服务器：
   ```bash
   npm start
   ```

## 🌐 访问系统

启动成功后，在浏览器中访问：
- **主页面**: http://localhost:3000/index.html
- **API状态**: http://localhost:3000/api/status

### 1. GLM-4 Plus 配置
- 访问：https://open.bigmodel.cn/
- 注册并获取API Key
- 在页面中输入API Key并保存

### 2. 通义千问Max 配置

- 开通服务并获取API Key
- 在页面中输入API Key并保存

## 🎯 使用流程

1. **启动系统** → 运行 start.bat
2. **配置API** → 在页面上方配置面板输入API密钥

## 🚨 常见问题解决

### Node.js 未安装
**症状**: 提示 'node' 不是内部或外部命令
**解决**: 下载安装 Node.js LTS 版本

### 端口被占用
**症状**: 启动时提示端口3000被占用
**解决**: 
1. 关闭占用端口的程序
2. 或修改 server.js 中的端口号

### API连接失败
**症状**: 配置后显示连接失败
**解决**:
1. 检查API密钥是否正确
2. 确认网络连接正常
3. 检查API服务商的服务状态

### 页面无法访问
**症状**: 浏览器显示无法访问
**解决**:
1. 确认服务器已启动
2. 检查防火墙设置
3. 尝试使用 127.0.0.1:3000 访问
